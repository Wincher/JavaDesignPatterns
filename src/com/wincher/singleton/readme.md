
## singleton describe
>在阎宏博士的《JAVA与模式》一书中开头是这样描述单例模式（singleton）模式的：
作为对象的创建模式，单例模式确保某一个类只有一个实例，而且自行实例化并向整个系统提供这个实例。这个类称为单例类。

单例模式的结构
　　单例模式的特点：  
单例类只能有一个实例。  
单例类必须自己创建自己的唯一实例。  
单例类必须给所有其他对象提供这一实例。
